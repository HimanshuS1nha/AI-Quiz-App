## Get started

1. Add Google Gemini API key in a .env file in the server folder.

2. Server

   ```bash
   npm install
   npm run dev
   ```

3. Client

   ```bash
    npm install
    npm run start
   ```

## Features

- Users can select a topic and practice on AI-generated MCQ questions about that topic
